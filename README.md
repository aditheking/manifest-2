TouchWiz PORT 6.0.X MSM8226
=========================


Getting Started
---------------

To initialize your local repository using the TW-LL-MSM8226 trees, use a command like this:

    repo init -u git://github.com/TW-MM-msm8226/manifest.git -b tw_mm

Then to sync up:

    repo sync

----------------
